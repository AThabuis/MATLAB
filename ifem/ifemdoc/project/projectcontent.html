Projects¶

The best way to learn iFEM is to do the following projects designed for Math 226: Numerical PDE in UC Irvine.

  • Finite Difference Methods in 1-D
  • Finite Difference Methods
  • Linear Finite Element Method
  • Fast Multipole Methods
  • Multigrid Methods
  • Nonlinear Poisson Boltzmann Equation
  • Adaptive FEM for PBE
  • Adaptive FEM for Heat Equation
  • Finite Element Methods for Stokes equations
  • MAC Scheme for Stokes equations
  • Fast Solvers for Stokes equations
  • Transient Navier-Stokes Equations
  • Edge element for Maxwell Equations
  • Wave equations
